## 📋 案例概述

本案是关于**建设项目压覆矿产资源补偿纠纷**的典型案例。原告郭某（某采矿实际经营者）诉被告某甲公司（张唐铁路建设单位）、某乙公司（张唐铁路代建单位），请求赔偿因张唐铁路建设压覆某采矿矿业权造成的各项损失。本案历经一审、重审、二审，最终由最高人民法院作出终审判决。

### 核心争议

1. **压覆范围**：案涉压覆系全部压覆抑或部分压覆？
2. **补偿范围与标准**：矿业权价值、竖井、巷道工程价值、资产补偿价值、前期费用和投入及停业损失等应如何认定？
3. **补偿款利息**：某甲公司是否应当支付补偿款利息？
4. **主体资格**：某乙公司是否为本案适格被告，应否与某甲公司共同承担给付案涉压覆补偿款责任？

## 📚 裁判要旨

最高人民法院在本案中明确了以下四项裁判规则：

### 一、建设项目未全部压覆但客观上导致采矿权范围内矿产资源全部不能开发利用的，应当视为全部压覆

**裁判理由**：
- 本案中，张唐铁路压覆某采矿矿区面积达95.04%，矿区内矿产资源不能继续开发利用，某采矿的矿业权不能正常行使。
- 某采矿因张唐铁路压覆导致其采矿权无法延续，实质上已无开采价值。
- **规则**：建设项目未全部压覆但客观上导致采矿权范围内矿产资源全部不能开发利用的，应当视为全部压覆。

### 二、涉及社会公共利益的铁路建设项目依法办理压覆审批的，压覆行为不具有违法性，不构成侵权；采矿权人因此遭受损失的，可以依法请求补偿

**裁判理由**：
- 张唐铁路系经国家发展和改革委员会批准立项的国家重大建设工程项目，具有社会公益性质。
- 某甲公司作为建设单位对沿线矿产资源压覆情况依法委托第三方进行了调查评估，并经过了自然资源主管部门的压覆审批。
- 某甲公司、某乙公司对案涉压覆行为不具有过错，不应承担侵权赔偿责任。
- **规则**：涉及社会公共利益的铁路建设项目依法办理压覆审批的，压覆行为不具有违法性，不构成侵权；采矿权人因此遭受损失的，可以依法请求补偿。

### 三、压覆补偿以直接损失补偿为原则，补偿范围包括采矿权价值、被压覆矿产资源分担的勘查投资、已建的开采设施投入和搬迁相应设施费用等直接损失

**裁判理由**：
- 原国土资源部137号文第四条第三项规定：“建设项目压覆已设置矿业权矿产资源的，新的土地使用权人还应同时与矿业权人签订协议，协议应包括矿业权人同意放弃被压覆矿区范围及相关补偿内容。补偿的范围原则上应包括：1.矿业权人被压覆资源储量在当前市场条件下所应缴的矿业权价款（无偿取得的除外）；2.所压覆的矿产资源分担的勘查投资、已建的开采设施投入和搬迁相应设施等直接损失。”
- 该规范性文件确立了矿业权压覆补偿范围原则上应包括的损失项目，尽管没有排除其他可能的损失，但蕴含着以直接损失为补偿的一般原则。
- **规则**：压覆补偿以直接损失补偿为原则，补偿范围包括采矿权价值、被压覆矿产资源分担的勘查投资、已建的开采设施投入和搬迁相应设施费用等直接损失。当事人请求基于被压覆矿产资源储量在压覆时市场条件下所应缴的矿业权价款或者出让收益确定采矿权价值损失的，人民法院应予支持。

### 四、被压覆矿产资源储量应当以自然资源主管部门压覆审批所依据的调查评估报告或者自然资源主管部门评审备案的剩余矿产资源储量为准

**裁判理由**：
- 矿产资源储量评审备案，是指自然资源主管部门落实矿产资源国家所有的法律要求、履行矿产资源所有者职责，依申请对申请人申报的矿产资源储量进行审查确认，纳入国家矿产资源实物账户，作为国家管理矿产资源重要依据的行政行为。
- 矿产资源储量评审备案制度一直延续至今，矿业权人未依法申报矿产资源储量评审备案，将导致相应矿产资源储量的真实性无法确认，进而将会产生对己不利的法律后果，其据此主张的民事权益将可能难以获得支持。
- **规则**：被压覆矿产资源储量应当以自然资源主管部门压覆审批所依据的调查评估报告或者自然资源主管部门评审备案的剩余矿产资源储量为准。采矿权人自行委托相关单位出具的矿产资源储量勘查或者核实报告，未依法报送自然资源主管部门评审备案的，人民法院不予采信。

## 🔑 核心知识点

### 一、压覆补偿的性质

| 要点 | 内容 |
|------|------|
| **行为性质** | 涉及社会公共利益的铁路建设项目依法办理压覆审批的，压覆行为不具有违法性，不构成侵权 |
| **补偿性质** | 采矿权人因此遭受损失的，可以依法请求补偿（而非赔偿） |
| **补偿原则** | 以直接损失补偿为原则 |
| **补偿范围** | 采矿权价值、被压覆矿产资源分担的勘查投资、已建的开采设施投入和搬迁相应设施费用等直接损失 |

### 二、压覆范围的认定

| 情形 | 认定规则 |
|------|----------|
| **全部压覆** | 建设项目全部压覆采矿权范围内矿产资源的，当然视为全部压覆 |
| **部分压覆但实质上导致全部不能开发利用** | 建设项目未全部压覆但客观上导致采矿权范围内矿产资源全部不能开发利用的，应当视为全部压覆 |
| **压覆范围的确定** | 以自然资源主管部门压覆审批所依据的调查评估报告为准 |

### 三、补偿范围与标准的认定

| 项目 | 认定规则 | 举证责任 |
|------|----------|----------|
| **矿业权价值损失** | 应当以被压覆矿产资源储量在压覆时市场条件下所应缴的矿业权价款或者出让收益确定 | 采矿权人应当提供自然资源主管部门评审备案的矿产资源储量报告 |
| **竖井、巷道等地下资产损失** | 应当根据压覆范围、工程施工实际情况、折旧情况等因素综合认定 | 采矿权人应当提供工程施工资料、评估报告等证据 |
| **地上资产补偿价值** | 应当根据资产实际价值、折旧情况等因素综合认定 | 采矿权人应当提供资产评估报告等证据 |
| **前期费用和投入及停业损失** | 应当根据实际投入、停业时间等因素综合认定 | 采矿权人应当提供相关费用凭证、评估报告等证据 |

### 四、矿产资源储量评审备案的效力

| 要点 | 内容 |
|------|------|
| **评审备案的性质** | 矿产资源储量评审备案，是指自然资源主管部门落实矿产资源国家所有的法律要求、履行矿产资源所有者职责，依申请对申请人申报的矿产资源储量进行审查确认，纳入国家矿产资源实物账户，作为国家管理矿产资源重要依据的行政行为 |
| **评审备案的效力** | 矿产资源储量评审备案制度一直延续至今，矿业权人未依法申报矿产资源储量评审备案，将导致相应矿产资源储量的真实性无法确认 |
| **未评审备案的后果** | 采矿权人自行委托相关单位出具的矿产资源储量勘查或者核实报告，未依法报送自然资源主管部门评审备案的，人民法院不予采信 |

### 五、压覆补偿责任的主体

| 主体 | 责任认定 |
|------|----------|
| **建设单位** | 根据《中华人民共和国矿产资源法》第三十三条、《中华人民共和国矿产资源法实施细则》第三十五条、原国土资源部137号文第四条等有关规定，建设项目压覆矿产资源的补偿责任主体应为工程项目的建设单位 |
| **代建单位** | 代建单位系受建设单位委托代建工程项目的单位，不是本案的补偿主体，不应对采矿权人承担补偿责任（除非代建合同约定代建单位承担补偿责任） |

## ⚡ 实务提示

### 一、对采矿权人的提示

1. **及时办理矿产资源储量评审备案**：在委托相关单位进行矿产资源储量勘查或者核实后，应当及时依法报送自然资源主管部门评审备案，否则人民法院不予采信
2. **保存好相关证据**：包括矿产资源储量报告、工程施工资料、资产评估报告、相关费用凭证等
3. **依法请求补偿**：在建设项目压覆矿产资源后，应当及时与建设单位协商补偿事宜；协商不成的，依法向人民法院提起诉讼

### 二、对律师的提示

1. **重点举证矿产资源储量评审备案文件**：在代理采矿权人提起压覆补偿诉讼时，应当重点举证自然资源主管部门评审备案的矿产资源储量报告
2. **正确区分赔偿与补偿**：涉及社会公共利益的建设项目依法办理压覆审批的，压覆行为不具有违法性，不构成侵权，应当请求补偿而非赔偿
3. **正确确定被告**：压覆补偿责任的主体应为工程项目的建设单位；代建单位一般不承担补偿责任

### 三、对人民法院的提示

1. **正确认定压覆范围**：建设项目未全部压覆但客观上导致采矿权范围内矿产资源全部不能开发利用的，应当视为全部压覆
2. **正确认定补偿范围**：压覆补偿以直接损失补偿为原则，补偿范围包括采矿权价值、被压覆矿产资源分担的勘查投资、已建的开采设施投入和搬迁相应设施费用等直接损失
3. **正确认定矿产资源储量**：被压覆矿产资源储量应当以自然资源主管部门压覆审批所依据的调查评估报告或者自然资源主管部门评审备案的剩余矿产资源储量为准
