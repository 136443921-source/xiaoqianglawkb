# templates/ 目录 - 页面模板

本目录存放 AI 生成 wiki/ 页面时参考的模板。

## 可用模板

- **concept-template.md** - 法律概念页模板
- **case-template.md** - 案例页模板
- **source-summary-template.md** - 源摘要页模板
- **comparison-template.md** - 对比分析页模板

## 使用说明

1. AI 编译时会参考这些模板生成页面
2. 你可以自定义模板格式，AI 会学习并遵循
3. 如果不需要模板，可以删除本目录（AI 会根据 AGENTS.md 规范生成页面）
