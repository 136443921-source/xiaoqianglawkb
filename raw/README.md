# raw/ 目录 - 原始资料存放处

本目录存放原始法律资料，包括：

- **articles/** - 法律文章、评论
- **judgments/** - 判决书、裁定书
- **papers/** - 学术论文
- **regulations/** - 法律法规、司法解释
- **notes/** - 个人笔记、思考
- **temp/** - 临时收集、待分类

## 使用说明

1. 将原始资料保存到对应子目录
2. 资料格式：Markdown (.md) 或 PDF
3. 建议添加元数据（参见 数据采集指南.md）

## 注意事项

- AI 编译时只读不修改本目录
- 本目录是事实来源，请勿随意删除
- 采集资料时遵循"宁缺毋滥"原则
