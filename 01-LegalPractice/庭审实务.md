## 一、庭审基本功核心能力

### 1.1 事实梳理能力

**目标**：快速、准确识别案件关键事实，构建清晰的事实框架。

**方法**：
- 制作《案件事实时间线》
- 识别争议焦点事实 vs. 无争议事实
- 标注证据支撑情况（有证据/无证据/证据不足）

**常见误区**：
- ❌ 陷入细节，忽略主线
- ❌ 事实与法律分析混淆
- ✅ 用时间线 + 争议焦点表梳理

---

### 1.2 证据组织能力

**目标**：构建完整、有说服力的证据链。

**证据清单结构**：
```
1. 证据名称
2. 证据来源（原件/复印件/电子证据）
3. 证明目的（每一份证据要证明什么）
4. 关联性（与争议焦点的对应关系）
```

**举证策略**：
- 按争议焦点分组举证
- 关键证据优先出示
- 预判对方质证意见，提前准备反驳

**常见误区**：
- ❌ 证据堆砌，无逻辑主线
- ❌ 忽略证据原件准备
- ✅ 证据目录 + 证据说明缺一不可

---

### 1.3 法律分析能力

**目标**：准确适用法律，构建法律论证框架。

**法律检索流程**：
1. 确定法律关系性质
2. 检索适用法律（法律、行政法规、司法解释）
3. 检索类案裁判规则（指导性案例、参考案例）
4. 分析法律适用的争议点

**法律意见书结构**：
- 案情摘要
- 争议焦点
- 法律分析（适用法律 + 类案参考）
- 结论与建议

---

### 1.4 庭审表达能力

**目标**：清晰、简洁、有说服力地表达观点。

**庭审表达原则**：
- **简洁**：一句话说清核心观点
- **逻辑**：事实 → 证据 → 法律 → 结论
- **尊重**：尊重法庭、对方、证人

**庭审语言技巧**：
- 使用法言法语，避免口语化
- 关键数字、日期、名称要准确
- 避免情绪化表达

## 二、庭审对话技巧

### 2.1 直接询问（主询问）

**目标**：引导证人说出有利事实，避免诱导性提问。

**技巧**：
- 开放式提问："请描述一下当时的情况？"
- 避免封闭式提问："你是不是在场上？"（这是诱导性的）
- 按时间顺序提问，帮助证人回忆

**示例**：
```
❌ 错误："你当时是不是看到被告在场上？"
✅ 正确："请描述一下2025年6月17日下午3点，你在场上看到了什么？"
```

---

### 2.2 交叉询问（反询问）

**目标**：削弱对方证人可信度，揭示矛盾。

**技巧**：
- 使用封闭式提问（ yes/no 问题）
- 聚焦于证人之前陈述的矛盾
- 避免开放式提问（给证人解释机会）

**示例**：
```
✅ "你之前在笔录中说你不在场上，今天你说你在场上，哪个是真的？"
✅ "你看清楚了吗？"（攻击感知能力）
```

---

### 2.3 异议处理

**常见异议理由**：
- 诱导性提问
- 无关问题
- 重复提问
- 猜测性提问

**异议表达模板**：
```
"审判长/法官，反对。对方的问题是诱导性的，违反了《民事诉讼法》司法解释关于询问证人的规定。"
```

## 三、训练方法

### 3.1 模拟庭审

**流程**：
1. 选择典型案例
2. 分配角色（法官、原告、被告、证人）
3. 全流程演练（开庭 → 举证质证 → 辩论 → 最后陈述）
4. 录像回放 + 复盘

**复盘要点**：
- 事实梳理是否清晰？
- 证据组织是否有逻辑？
- 法律分析是否准确？
- 庭审表达是否简洁有力？

---

### 3.2 案例研习

**步骤**：
1. 阅读判决书全文
2. 提取法院裁判规则
3. 分析胜败诉原因
4. 总结可复用策略

**推荐资源**：
- 人民法院案例库（指导性案例、参考案例）
- 中国裁判文书网
- 最高法发布的典型案例

---

### 3.3 导师反馈

**关键**：
- 主动请求导师观摩庭审
- 录音/录像自我复盘
- 定期提交庭审心得给导师审阅

## 四、实战建议（来自员额法官反馈）

### 4.1 法官关注什么？

基于员额法官 algamott 的反馈：

1. **事实清楚**：法官需要快速理解案件事实
2. **证据充分**：证据链完整，无重大遗漏
3. **法律准确**：适用法律正确，类案参考恰当
4. **表达高效**：不浪费法庭时间

**关键启示**：
- 庭前准备至关重要
- 用图表、时间线辅助说明复杂事实
- 提前准备《庭审提纲》

---

### 4.2 常见法官提问

**事实类**：
- "原告/被告，你们的合同是怎么约定的？"
- "这个款项是什么性质？"

**证据类**：
- "这份证据的原件在哪里？"
- "这份证据与本案有什么关系？"

**法律类**：
- "你们主张的法律依据是什么？"
- "有没有相关类案？"

**应对策略**：
- 提前准备《法官可能提问清单》
- 与当事人充分沟通，确保事实陈述一致
