## 📋 制度概述

**《民间非营利组织会计制度》**是规范民间非营利组织会计核算的规范性文件。

### 立法目的

为了规范民间非营利组织的会计核算，保证会计信息质量，根据《中华人民共和国会计法》及国家其他有关法律、行政法规的规定，制定本制度。

### 适用范围

本制度适用于在中华人民共和国境内依法设立的符合本制度规定特征的民间非营利组织。

**包括**：
- 社会团体
- 基金会
- 社会服务机构
- 宗教活动场所
- 国际性社会团体
- 外国商会
- 境外非政府组织在中国境内依法登记设立的代表机构

### 民间非营利组织特征

适用本制度的民间非营利组织应当同时具备以下特征：

1. **为公益目的或者其他非营利目的成立**
2. **资源提供者向该组织投入资源不取得经济回报**
3. **资源提供者对该组织的财产不保留或享有任何财产权利**

## 🔑 核心会计核算原则

### 1. 会计核算基础

**权责发生制**：民间非营利组织应当以权责发生制为基础。

### 2. 会计核算基本原则

| 原则 | 说明 |
|------|------|
| **客观性原则** | 以实际发生的交易或者事项为依据，如实反映财务状况、业务活动情况和现金流量 |
| **相关性原则** | 所提供的会计信息应当能够满足会计信息使用者的需要 |
| **实质重于形式原则** | 按照交易或者事项的实质进行核算，而不应当仅仅按照它们的法律形式作为依据 |
| **一致性原则** | 会计政策前后各期应当保持一致，不得随意变更 |
| **可比性原则** | 会计核算应当按照规定的会计处理方法进行，会计信息应当口径一致、相互可比 |
| **及时性原则** | 会计核算应当及时进行，不得提前或延后 |
| **明晰性原则** | 会计核算和编制的财务会计报告应当清晰明了，便于理解和使用 |
| **配比原则** | 所发生的费用应当和与其相关的收入相配比 |
| **谨慎性原则** | 会计核算应当遵循谨慎性原则 |
| **划分收益性支出与资本性支出原则** | 合理划分应当计入当期费用的支出和应当予以资本化的支出 |
| **重要性原则** | 对资产、负债、净资产、收入、费用等有较大影响的重要会计事项，必须按照规定的会计方法和程序进行处理 |

### 3. 会计期间

会计核算应当划分会计期间，分期结算账目和编制财务会计报告。

- **会计期间**：至少分为年度和月度
- **起讫日期**：采用公历日期

### 4. 记账本位币

会计核算应当以人民币作为记账本位币。

**特殊情况**：
- 业务收支以人民币以外的货币为主的民间非营利组织，可以选定其中一种货币作为记账本位币
- 编制的财务会计报告应当折算为人民币

### 5. 外币业务

**外币账户包括**：
- 外币现金
- 外币银行存款
- 以外币结算的债权和债务账户

**核算原则**：
1. 发生外币业务时，应当将有关外币金额折算为记账本位币金额记账
2. 除另有规定外，所有与外币业务有关的账户，应当采用业务发生当日的即期汇率
3. 各种外币账户的外币余额，期末时应当按照期末的即期汇率折合为记账本位币
4. 按照期末的即期汇率折合的记账本位币金额与账面记账本位币金额之间的差额，作为汇兑损益计入当期费用

**例外情况**：
属于在借款费用应予资本化的期间内发生的与购建固定资产、无形资产等非流动资产有关的外币专门借款本金及其利息所产生的汇兑差额，应当予以资本化，计入固定资产、无形资产等非流动资产成本。
