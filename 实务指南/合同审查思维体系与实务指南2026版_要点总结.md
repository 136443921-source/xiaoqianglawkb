## 二、核心要点

### 2.1 合同审查的两个核心目的
1. **防范风险**：识别和防范合同中的法律风险
2. **实现交易**：确保合同能够实现委托人的交易目的

### 2.2 合同审查的三个基本前提
1. **委托人交易目的**：了解委托人为什么要签订这份合同
2. **委托人谈判地位**：确定委托人在交易中的优势/劣势地位
3. **合同交易模式**：理解合同的交易结构和商业模式

### 2.3 合同审查的四个步骤（由大及小、层层递进）
1. **审查交易模式**：
   - 以合同主体为基本要素
   - 参照买卖合同的模式梳理主体之间的权利义务
   - 提炼商业交易模式
2. **审查合同结构**：
   - 按照"主体、正文、签署"三要素原则
   - 审查合同条款之间的逻辑性
   - 审查合同条款的先后顺序和表述方式
3. **审查合同条款**（核心）：
   - 逐条审查合同条款
   - 同时修改文字和标点符号
   - 补增缺项条款
4. **审查文字符号**：
   - 审查合同条款的文字表达
   - 审查标点符号的使用

### 2.4 合同审查的五个标准
1. **合法性**：合同条款符合法律法规规定
2. **完备性**：合同条款完整、全面
3. **可操作性**：合同条款具有可操作性，能够实际执行
4. **严谨性**：合同条款表述严谨、准确
5. **美观性**：合同格式美观、规范

### 2.5 合同审查的六个提升途径
1. **加强法律知识学习**：学习法律法规、司法解释、司法裁判规则
2. **积累法律实践经验**：积累客户类似交易的经验、类似合同案件的司法裁判规则
3. **培养责任心**：充分意识到合同对于客户经营的重要性
4. **掌握审查方法**：掌握合同审查的常用方法（补增、删改、建议、提示等）
5. **多审查合同**：通过实践提升审查水平
6. **持续总结反思**：持续积累、保持思考、不断总结

## 三、实务提示

### 3.1 如何应用思维体系
1. **拿到待审合同，不要直接修改**：
   - 首先**通读合同三遍以上**（复杂疑难合同需要通读更多遍数）
   - 通过整体通读了解交易模式、合同结构
   - 再对合同条款进行修改

2. **避免"当局者迷"**：
   - 不要陷入既定的合同文本审查之中
   - 要有"由大及小"的思维方式
   - 既要审查合同整体的完整性和全面性，又要审查具体条款的准确性和可操作性

3. **修改模式的选择**：
   - **清洁修改模式**：直接修改合同文本，不保留修改痕迹
   - **痕迹修改模式**：保留修改痕迹（如Word的"修订"功能）
   - **意见书模式**：不直接修改合同，而是出具合同审查意见书
   - **以客户的需求为中心**，委托人希望以哪种模式修改就采取哪种模式

### 3.2 合同审查的常见问题
1. **顾此失彼**：审查深度不够，有的仅起到校对的作用
2. **前后不一**：合同条款前后不一致、重复啰嗦
3. **缺乏整体性**：思维被合同文本"牵着走"，"跳"不出来
4. **修改尺度不当**：没有充分考虑委托人的谈判地位

为了便于保存和归档，建议用**"日期+文件名称+修改人+版本"**的模式进行命名。
