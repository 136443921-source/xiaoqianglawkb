# 王德明担保合同纠纷案

> 核心突破口：保证合同签名/指纹系伪造 → 担保合同不发生效力。多线并行策略：执行异议 + 司法鉴定 + 再审/检察监督 + 金融监管投诉 + 刑事报案。

## 一、案件信息

- **案号**：(2018) 黔0330民初1497号
- **当事人**：习水县农村信用合作联社（原告）vs 王方利（借款人）、陈彬、宇敏、王德明、侯天香（保证人）
- **背景**：2014年王方利以胜利砖厂名义贷款50万元，罗肖（信用社员工）办理。王德明、侯天香称《保证合同》签名/指纹系伪造
- **2018判决**：王方利偿还本息，保证人连带，信用社对抵押房产优先受偿。王德明等未到庭。
- **2025-04-15**：王德明账户再次冻结，发现贷款未清、已核销表外，卷宗《保证合同》与2018年信用社版本完全不同

## 二、核心争议焦点

1. **保证合同签名/指纹是否伪造** → 若伪造则担保无效（《民法典》143、490条）
2. **2018诉讼程序违法**：送达违规、主要证据伪造、当事人因信任"已解决"未出庭
3. **时效误区**：签名伪造涉合同效力根本问题，诉讼时效/抗辩时效不适用
4. **核销表外≠债权消灭**：账销案存但若担保伪造则根本无有效担保
