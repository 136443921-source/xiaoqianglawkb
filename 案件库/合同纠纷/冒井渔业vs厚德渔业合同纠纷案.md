# 冒井渔业vs厚德渔业合同纠纷案

> 二审终审判决，部分改判。再审可能性评估为C级（谨慎申请）。

## 一、案件信息

- **案号**：(2022) 黔26民终1384号
- **当事人**：贵州省冒井冷泉水渔业有限公司（上诉人/原审原告）vs 贵州省厚德鳝鱼生态渔业有限公司（被上诉人/原审被告）
- **案由**：合同纠纷——《退出补偿协议》履行争议
- **审理法院**：一审-岑巩县人民法院；二审-黔东南中院
- **判决结果**：一审驳回全部诉请 → 二审部分改判：支持第一期12.5万元及律师费2万元，驳回第二、三、四期诉请

## 二、核心争议焦点

1. 协议是否已解除（协商解除函 ≠ 单方解除通知，《民法典》565条）
2. 第二、三、四期付款条件是否成就（举证责任在主张条件成就方）
3. 厚德公司是否存在恶意阻碍条件成就（《民法典》159条未适用）

## 三、裁判规则提取

| 规则 | 内容 |
|------|------|
| R001 | "协商解除函"不构成单方解除通知，协议未因此解除 |
| R002 | 附条件付款协议中，主张条件成就方负举证责任 |
| R003 | 环评未通过导致项目无法推进，构成付款条件未成就的正当理由 |
| R004 | 协议有效且未解除时，付款条件未成就的不支持付款请求 |

## 四、经验提炼

- 本案二审改判仅支持第一期，核心败因在于冒井公司未能举证证明第二、三、四期付款条件已成就
- 再审最大障碍：已超6个月申请期限
- 替代方案：执行和解优先于再审
