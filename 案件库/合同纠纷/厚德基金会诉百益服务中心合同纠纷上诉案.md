# 厚德基金会诉百益服务中心合同纠纷上诉案

> 核心争点：一审法院以"先刑后民"裁定驳回起诉是否适用法律错误？马婵娟涉嫌经济犯罪是否影响本案民事审理？

## 一、案件信息

- **案号**：(2025) 黔0325民初230号之一（一审）
- **当事人**：贵州省厚德公益基金会（上诉人）vs 道真仡佬族苗族自治县百益社会工作服务中心、李丽、李佳豪（被上诉人）
- **案由**：合同纠纷——百益服务中心未按协议约定用途使用290,000元项目资金，法定代表人李丽将200,000元转入案外个人账户
- **一审裁定**：驳回起诉（认定须以另一案审理结果为依据）

## 二、核心争议焦点

1. **程序争议**：本案是否存在经济犯罪嫌疑？是否应"先刑后民"？
2. **事实争议**：百益服务中心收到29万元后是否存在挪用行为？
3. **法律争议**：案外人马婵娟涉嫌犯罪是否影响本案民事审理？
4. **责任争议**：李丽、李佳豪是否应承担连带责任（法人人格否认）？
5. **金额争议**：已支出的43,864.17元是否应扣除？

## 三、我方核心上诉理由

1. 本案是典型民事合同纠纷，一审错误适用"先刑后民"规则
2. 根据《九民会议纪要》第105条，民事案件与刑案有关联但非同一法律事实的，应继续审理
3. 最高法(2025)民再164号裁定：不应当仅因关联刑案未审结就驳回起诉
4. 李丽、李佳豪构成人格混同+财产混同，应承担连带责任（《公司法》第20条第3款、《民法典》第83条第2款）
