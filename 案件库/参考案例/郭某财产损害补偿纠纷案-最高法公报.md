# 郭某与某甲公司等财产损害补偿纠纷案（最高法公报）

> 最高人民法院公报案例。确立四项压覆矿产资源补偿核心裁判规则。

## 一、案例概要

- **案号**：(2024) 最高法民终28号
- **当事人**：郭某（采矿实际经营者）vs 某甲公司（张唐铁路建设单位）、某乙公司（代建单位）
- **核心争议**：压覆范围（全部/部分）、补偿范围与标准、补偿款利息、代建单位责任主体资格

## 二、四项裁判要旨

### 规则一：未全部压覆但客观上导致全部不能开发利用 → 视为全部压覆
- 张唐铁路压覆矿区面积95.04%，采矿权无法延续，实质上已无开采价值

### 规则二：涉及社会公共利益的建设项目依法办理压覆审批 → 不构成侵权，只补偿不赔偿
- 压覆行为不具有违法性，采矿权人可依法请求补偿（非赔偿）
- **区分补偿与赔偿**：补偿=合法行为造成的损失填补；赔偿=违法行为造成的损害填补

### 规则三：压覆补偿以直接损失补偿为原则
- 范围：采矿权价值 + 勘查投资分担 + 已建开采设施投入 + 搬迁费用
- 采矿权价值 = 被压覆资源储量在压覆时市场条件下所应缴的矿业权价款

### 规则四：矿产资源储量以自然资源主管部门评审备案为准
- 自行委托的报告未经评审备案 → 法院不予采信
- 评审备案是行政行为，未备案的储量真实性无法确认

## 三、实务价值

- 对律师：正确区分补偿vs赔偿，举证重点在评审备案文件
- 对采矿权人：及时办理矿产资源储量评审备案
- 对建设单位：补偿主体=建设单位，代建单位一般不承担

## 四、关联法条

- 《矿产资源法》第33条
- 《矿产资源法实施细则》第35条
- 国土资发〔2010〕137号文第4条
