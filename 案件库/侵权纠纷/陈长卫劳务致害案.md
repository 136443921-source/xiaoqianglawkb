# 陈长卫劳务致害案

> 关键改判：二审推翻一审"三牌更换属于原分包范围"的事实认定，适用《民法典》第1239条让供电局承担高压电无过错责任。

## 一、案件信息

- **案号**：(2023) 黔03民终5221号（二审）；(2023) 黔0302民初1174号（一审）
- **当事人**：陈长卫 vs 遵义光明电力实业有限公司、遵义供电局、润通公司、王远飞、陈宗元
- **案由**：提供劳务受害者责任纠纷——带电线路杆塔作业被高压电击伤
- **判决结果**：二审改判：供电局20%、陈长卫20%、光明公司等连带60%

## 二、核心争议焦点

1. "三牌更换"是否属于原分包范围（核心事实争议）
2. 供电局作为高压电经营者是否承担无过错责任（《民法典》1239条）
3. 违法分包链条上各方责任分配

## 三、裁判规则提取

| 规则 | 内容 |
|------|------|
| R001 | 高压电经营者承担无过错责任，不因发包关系免除（《民法典》1239条） |
| R002 | "重新立项"视为新合同关系，不属于原分包范围 |
| R003 | 违法分包+违法转包+雇主无资质 → 各方承担连带责任 |
| R004 | 被侵权人有过错的，过失相抵减轻侵权人责任（《民法典》1173条） |
| R005 | 一人公司法定代表人不能证明公司财产独立 → 连带责任 |

## 四、经验提炼

- 关键证据：《35kV线路工程后续工作报告》证明润通公司"无力完成剩余三牌挂设"→推翻一审事实认定
- 一审采信的《更换三牌情况说明》为光明公司单方出具，与后续事实矛盾 → 二审不予采信
- 高压电案件：锁定"高压电经营者"身份 → 适用第1239条无过错责任
