## 📋 民事诉讼法概述

《民事诉讼法》是规范民事诉讼程序的基本法律，共 4 编、29 章、308 条，涵盖从立案到执行的全流程。

### 适用范围

- 公民之间、法人之间、其他组织之间以及他们相互之间因财产关系和人身关系提起的民事诉讼
- 在中华人民共和国领域内进行民事诉讼，必须遵守本法

## 🔑 核心知识点

### 一、管辖（第2章）

| 管辖类型 | 法条 | 要点 |
|----------|------|------|
| 一般管辖 | 第22条 | 对公民提起的民事诉讼，由被告住所地人民法院管辖 |
| 特殊管辖 | 第23-32条 | 合同纠纷（被告住所地/合同履行地）、侵权责任（被告住所地/侵权行为地） |
| 专属管辖 | 第33-34条 | 不动产纠纷（不动产所在地）、港口作业（港口所在地）、继承遗产（被继承人死亡时住所地/主要遗产所在地） |
| 协议管辖 | 第35条 | 合同或者其他财产权益纠纷的当事人可以书面协议选择被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地等与争议有实际联系的地点的人民法院管辖（不得违反本法对级别管辖和专属管辖的规定） |
| 移送管辖 | 第36条 | 人民法院发现受理的案件不属于本院管辖的，应当移送有管辖权的人民法院 |
| 指定管辖 | 第37条 | 有管辖权的人民法院由于特殊原因，不能行使管辖权的，由上级人民法院指定管辖 |

### 二、证据（第6章）

- **证据类型**（第66条）：当事人的陈述、书证、物证、视听资料、电子数据、证人证言、鉴定意见、勘验笔录
- **举证责任**（第67条）：当事人对自己提出的主张，有责任提供证据（谁主张谁举证）
- **举证责任倒置**（第68-73条）：特殊侵权案件（如环境污染、医疗损害等）由被告举证
- **人民法院调查收集证据**（第67条第2款）：当事人及其诉讼代理人因客观原因不能自行收集的证据，或者人民法院认为审理案件需要的证据，人民法院应当调查收集
- **证据保全**（第84条）：在证据可能灭失或者以后难以取得的情况下，当事人可以在诉讼过程中向人民法院申请保全证据，人民法院也可以主动采取保全措施

### 三、审判程序（第12-20章）

| 程序 | 法条 | 要点 |
|------|------|------|
| 起诉条件 | 第122条 | 原告与本案有直接利害关系 + 有明确被告 + 有具体诉讼请求和事实理由 + 属于法院受理范围和受诉法院管辖 |
| 受理期限 | 第126条 | 符合起诉条件的，应当在7日内立案并通知当事人；不符合的，应当在7日内作出裁定书不予受理 |
| 送达 | 第85-92条 | 直接送达、留置送达、电子送达、委托送达、邮寄送达、转交送达、公告送达 |
| 开庭前准备 | 第128-133条 | 送达起诉状副本、被告答辩、交换证据、庭前会议 |
| 开庭审理 | 第134-146条 | 开庭公告、法庭调查、法庭辩论、最后陈述、评议宣判 |
| 简易程序 | 第160-164条 | 事实清楚、权利义务关系明确、争议不大的简单民事案件，可以适用简易程序（3个月内审结） |
| 上诉期限 | 第171条 | 当事人不服地方人民法院第一审判决的，有权在判决书送达之日起15日内向上一级人民法院提起上诉；不服裁定的，有权在裁定书送达之日起10日内提起上诉 |
| 再审程序 | 第206-213条 | 当事人对已经发生法律效力的判决、裁定，认为有错误的，可以向上一级人民法院申请再审（不停止判决、裁定的执行） |

### 四、执行程序（第21-24章）

- **申请执行期限**（第246条）：申请执行的期间为2年（从法律文书规定履行期间的最后一日起计算）
- **执行措施**（第248-264条）：查询、扣押、冻结、划拨、变价被执行人的财产；扣留、提取被执行人应当履行义务部分的收入；查封、扣押、拍卖、变卖被执行人应当履行义务部分的财产
- **失信被执行人**（第262条）：被执行人不履行法律文书确定的义务的，人民法院可以对其采取或者通知有关单位协助采取限制出境，在征信系统记录、通过媒体公布不履行义务信息以及法律规定的其他措施

## ⚡ 实务提示

1. **起诉前准备**：确认被告身份信息（姓名/名称、住所）、收集证据、确定管辖法院
2. **诉讼时效**：普通诉讼时效为3年（民法典第188条），从知道或应当知道权利受损以及义务人之日起计算
3. **证据保全**：在证据可能灭失或者以后难以取得的情况下，应当在起诉前或起诉后及时申请证据保全
4. **财产保全**：在起诉前或起诉后，可以申请对对方财产进行保全，防止对方转移财产
5. **上诉期限**：不服一审判决的，应当在15日内提起上诉；不服裁定的，应当在10日内提起上诉（务必注意期限）
6. **申请执行期限**：申请执行的期间为2年，从法律文书规定履行期间的最后一日起计算（务必在期限内申请执行）
