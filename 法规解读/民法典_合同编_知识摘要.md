## 📋 合同编概述

《民法典》第三编"合同"共 3 个分编、29 章、526 条，是合同法领域的核心规范，替代原《合同法》。

### 适用范围

- 平等主体之间的合同关系
- 身份关系协议（婚姻、收养、监护）优先适用专门规定，无规定时可参照适用

## 🔑 核心知识点

### 一、合同的订立（第2章）

| 概念 | 法条 | 要点 |
|------|------|------|
| 要约 | 第472条 | 内容具体确定 + 表明经受要约人承诺即受约束 |
| 要约邀请 | 第473条 | 拍卖公告、招标公告、商业广告等为要约邀请 |
| 承诺 | 第479条 | 受要约人同意要约的意思表示 |
| 合同成立时间 | 第483条 | 承诺生效时合同成立（另有规定或约定除外） |
| 采用书面形式 | 第490条 | 未采用书面形式但一方已履行主要义务且对方接受，合同成立 |

### 二、合同的效力（第3章）

| 效力状态 | 法条 | 情形 |
|----------|------|------|
| 有效 | 第143条 | 行为人有相应民事行为能力 + 意思表示真实 + 不违反法律/公序良俗 |
| 无效 | 第144、146、153、154条 | 无民事行为能力人实施；虚假意思表示；违反强制性规定/公序良俗；恶意串通 |
| 可撤销 | 第147-151条 | 重大误解；欺诈；胁迫；显失公平 |
| 效力待定 | 第145、171条 | 限制民事行为能力人实施；无权代理 |

### 三、合同的履行（第4章）

- **全面履行原则**（第509条）：按约定全面履行义务
- **抗辩权**（第525-527条）：
  - 同时履行抗辩权（第525条）
  - 先履行抗辩权（第526条）
  - 不安抗辩权（第527条）
- **债权人代位权**（第535条）：债务人怠于行使其债权，影响债权人债权实现的，债权人可向法院请求以自己名义代位行使
- **债权人撤销权**（第538-542条）：债务人无偿/低价处分财产，影响债权人债权实现的，债权人可请求撤销

### 四、合同的解除（第5章）

| 解除方式 | 法条 | 要点 |
|----------|------|------|
| 约定解除 | 第562条 | 当事人协商一致，可解除合同 |
| 约定解除权 | 第562条第2款 | 约定一方解除合同的事由，事由发生时解除权人可解除合同 |
| 法定解除 | 第563条 | 因不可抗力/预期违约/迟延履行等致使不能实现合同目的 |
| 解除程序 | 第565条 | 通知解除，对方有异议可请求法院/仲裁机构确认 |
| 解除后果 | 第566条 | 尚未履行的终止履行；已履行的可请求恢复原状/采取其他补救措施并有权请求赔偿损失 |

### 五、违约责任（第8章）

- **违约形态**：不履行、迟延履行、瑕疵履行、拒绝履行
- **违约责任形式**（第577条）：继续履行、采取补救措施、赔偿损失、支付违约金
- **定金罚则**（第586-587条）：定金不得超过主合同标的额的20%；给付定金一方违约无权请求返还，收受定金一方违约应双倍返还
- **违约金调整**（第585条）：约定的违约金低于损失的，可请求增加；过分高于损失的，可请求适当减少

### 六、典型合同重点（第二分编）

| 合同类型 | 重点法条 | 实务要点 |
|----------|----------|----------|
| 买卖合同 | 第595-647条 | 所有权保留（第641条）、风险负担（第604条） |
| 借款合同 | 第667-680条 | 利率限制（第680条：禁止高利放贷） |
| 租赁合同 | 第703-734条 | 买卖不破租赁（第725条）、优先购买权（第726条） |
| 建设工程合同 | 第788-808条 | 转包禁止（第791条）、优先受偿权（第807条） |
| 委托合同 | 第919-936条 | 任意解除权（第933条） |

## ⚡ 实务提示

1. **合同审查要点**：主体资格 → 意思表示真实性 → 合同条款完整性 → 违约责任明确性 → 争议解决条款
2. **定金 vs 订金**：定金适用罚则，订金不适用
3. **违约金过高标准**：一般不超过损失的30%（司法解释）
4. **合同解除异议期**：对方有异议的，应在约定期限或3个月内向法院/仲裁机构提出
5. **诉讼时效**：普通诉讼时效为3年（第188条），自知道或应当知道权利受损以及义务人之日起计算
