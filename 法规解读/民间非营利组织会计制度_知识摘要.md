## 一、概述

### 1.1 适用范围

本制度适用于在中华人民共和国境内依法设立的符合本制度规定特征的**民间非营利组织**。

**适用组织类型**（第2条）：
1. 社会团体
2. 基金会
3. 社会服务机构
4. 宗教活动场所
5. 国际性社会团体
6. 外国商会
7. 境外非政府组织在中国境内依法登记设立的代表机构等组织

**民间非营利组织应当同时具备以下特征**（第2条）：
1. 为公益目的或者其他非营利目的成立；
2. 资源提供者向该组织投入资源不取得经济回报；
3. 资源提供者对该组织的财产不保留或享有任何财产权利。

**不适用范围**：
- 企业（适用《企业会计准则》）
- 事业单位（适用《事业单位会计制度》）
- 行政机关（适用《行政单位会计制度》）

## 二、核心知识点

### 2.1 会计核算基本原则（第8条）

1. **可靠性**：会计核算应当以实际发生的交易或者事项为依据，如实反映民间非营利组织的财务状况、业务活动情况和现金流量等信息。
2. **相关性**：会计核算所提供的信息应当能够满足会计信息使用者（如出资人、设立人、捐赠人、会员、监管者等）的需要。
3. **实质重于形式**：会计核算应当按照交易或者事项的实质进行，而不应当仅仅按照它们的法律形式作为其依据。
4. **一致性**：会计政策前后各期应当保持一致，不得随意变更。
5. **可比性**：会计核算应当按照规定的会计处理方法进行，会计信息应当口径一致、相互可比。
6. **及时性**：会计核算应当及时进行，不得提前或延后。
7. **可理解性**：会计核算和编制的财务会计报告应当清晰明了，便于理解和使用。
8. **配比原则**：所发生的费用应当和与其相关的收入相配比。
9. **谨慎性**：会计核算应当遵循谨慎性原则。
10. **重要性**：会计核算应当遵循重要性原则。

### 2.2 资产（第14-20条）

#### 2.2.1 资产定义与分类（第14条）

**定义**：资产是过去的交易或者事项形成并由民间非营利组织拥有或者控制的资源，该资源预期会给民间非营利组织带来经济利益或者服务潜力。

**分类**：
1. **流动资产**：预期可在1年以内（含1年）变现或者耗用的资产，主要包括货币资金、短期投资、应收款项、预付账款、存货、待摊费用等。
2. **非流动资产**：长期投资、固定资产、无形资产等。
3. **受托代理资产**：民间非营利组织接受委托方委托从事受托代理业务而收到的资产。

#### 2.2.2 资产确认条件（第15条）

符合资产定义的资源，在同时满足以下条件时，应当确认为资产：
1. 与该资源有关的服务潜力很可能实现或者经济利益很可能流入民间非营利组织；
2. 该资源的成本或者价值能够可靠地计量。

#### 2.2.3 资产计量（第16条）

资产在取得时应当按照**实际成本**计量，但本制度有特别规定的，按照特别规定的计量基础进行计量。

**特别规定**（捐赠资产取得计量，第18条）：
1. 如果捐赠方提供了有关凭据（如发票、报关单、有关协议等）的，应当按照凭据上标明的金额作为入账价值。
2. 如果捐赠方没有提供有关凭据，或者凭据上标明的金额与受赠资产公允价值相差较大，受赠资产应当以其公允价值作为入账价值。
3. 对于民间非营利组织接受的固定资产、无形资产捐赠，如果捐赠方没有提供有关凭据，且有确凿的证据表明该资产的公允价值确实无法可靠计量，应当按照**名义金额（即人民币1元）**入账。
4. 对于民间非营利组织接受的服务捐赠，如果捐赠方提供了发票等有关凭据，且凭据上标明的金额能够反映受赠服务的公允价值，民间非营利组织应当按照凭据金额入账，其他情况不予确认。

#### 2.2.4 公允价值确定（第19条）

**定义**：公允价值是指市场参与者在计量日发生的有序交易中，出售一项资产所能收到或者转移一项负债所需支付的价格。

**确定顺序**：
1. 如果同类或者类似资产存在活跃市场的，应当按照同类或者类似资产的市场价格确定公允价值。
2. 如果同类或类似资产不存在活跃市场，或者无法找到同类或者类似资产的，应当采用合理的计价方法确定资产的公允价值。

#### 2.2.5 资产减值准备（第17条）

民间非营利组织应当定期或者至少于每年年度终了，对短期投资、应收款项、存货、长期投资等资产是否发生了减值进行检查，如果这些资产发生了减值，应当计提减值准备，确认减值损失，并计入当期费用。

对于固定资产、无形资产等其他资产，如果发生了重大减值，也应当计提减值准备，确认减值损失，并计入当期费用。

### 2.3 负债（第21-25条）

#### 2.3.1 负债定义与分类

**定义**：负债是过去的交易或者事项形成的现时义务，履行该义务预期会导致含有经济利益或者服务潜力的资源流出民间非营利组织。

**分类**：
1. **流动负债**：预计在1年以内（含1年）偿还的负债，主要包括短期借款、应付款项、应付工资、应交税金等。
2. **长期负债**：预计在1年以上偿还的负债，主要包括长期借款、长期应付款等。
3. **受托代理负债**：民间非营利组织因从事受托代理业务而发生的负债。

#### 2.3.2 负债确认条件

符合负债定义的义务，在同时满足以下条件时，应当确认为负债：
1. 与该义务有关的服务潜力很可能减少或者经济利益很可能流出民间非营利组织；
2. 未来流出的经济利益的金额能够可靠地计量。

### 2.4 净资产（第26-30条）

#### 2.4.1 净资产分类（第26条）

民间非营利组织的净资产应当按照其**是否受到限制**，分为以下两类：

1. **限定性净资产**：民间非营利组织资源提供者或者国家有关法律、行政法规对资产或资产所产生的经济利益的使用设置了限制，包括时间限制和（或）用途限制。
2. **非限定性净资产**：民间非营利组织净资产中除了限定性净资产之外的其他净资产。

**实务要点**：
- **时间限制**：资源提供者或者国家法律、行政法规规定所捐赠资产只能在某一特定时间之后使用
- **用途限制**：资源提供者或者国家法律、行政法规规定所捐赠资产必须用于某一特定用途
- 限定性净资产应当在资产负债表上单独列示
- 如果限定性净资产的限制已经解除，应当对净资产进行重新分类，将限定性净资产转为非限定性净资产

#### 2.4.2 净资产变动（第27-30条）

1. **期末结转**：
   - 期末，将各项收入结转至净资产（非限定性收入结转至非限定性净资产，限定性收入结转至限定性净资产）
   - 期末，将各项费用结转至非限定性净资产

2. **限定性净资产限制解除**（第30条）：
   - 当限定性净资产的相关限制得到解除时，应当对净资产进行重新分类，将限定性净资产转为非限定性净资产
   - 限制解除情形：
     - 时间限制已满
     - 用途限制已实现
     - 资源提供者或者国家法律、行政法规撤销了限制

### 2.5 收入（第31-40条）

#### 2.5.1 收入定义与分类（第31条）

**定义**：收入是指民间非营利组织开展业务活动取得的、导致本期净资产增加的经济利益或者服务潜力的流入。

**分类**：
1. **捐赠收入**：接受其他单位或者个人无偿给予的货币资金、实物资产等。
2. **会费收入**：社会团体按照章程等规定向会员收取的会费。
3. **提供服务收入**：民间非营利组织根据章程等规定向其服务对象提供服务取得的收入。
4. **政府补助收入**：接受政府拨款或者政府机构给予的补助。
5. **商品销售收入**：民间非营利组织销售商品等所形成的收入。
6. **投资收益**：民间非营利组织因对外投资取得的收益。
7. **其他收入**：除上述收入之外的其他收入。

#### 2.5.2 收入确认原则（第32-40条）

1. **捐赠收入确认**（第33条）：
   - 民间非营利组织对于无偿给予的资产，应当在同时满足以下条件时确认收入：
     - 资产的所有权转移给民间非营利组织；
     - 民间非营利组织没有义务向捐赠方提供商品或者劳务。
   - 捐赠收入应当分别按照**限定性收入**和**非限定性收入**进行核算

2. **会费收入确认**（第34条）：
   - 会费收入应当在同时满足以下条件时确认：
     - 与会员签订会费协议；
     - 会员会费已经收取或者能够合理估计。

3. **提供服务收入确认**（第35条）：
   - 提供服务收入应当在同时满足以下条件时确认：
     - 提供服务交易的结果能够可靠估计；
     - 与交易相关的经济利益能够流入民间非营利组织；
     - 相关的收入和成本能够可靠计量。

#### 2.5.3 收入核算科目设置

1. **捐赠收入**：设置"捐赠收入"科目，并按照是否存在限定分别按照"限定性收入"和"非限定性收入"进行明细核算
2. **会费收入**：设置"会费收入"科目
3. **提供服务收入**：设置"提供服务收入"科目
4. **政府补助收入**：设置"政府补助收入"科目，并按照是否存在限定分别按照"限定性收入"和"非限定性收入"进行明细核算
5. **商品销售收入**：设置"商品销售收入"科目
6. **投资收益**：设置"投资收益"科目
7. **其他收入**：设置"其他收入"科目

### 2.6 费用（第41-50条）

#### 2.6.1 费用定义与分类（第41条）

**定义**：费用是指民间非营利组织为开展业务活动所发生的、导致本期净资产减少的经济利益或者服务潜力的流出。

**分类**：
1. **业务活动成本**：民间非营利组织为了实现其业务活动目标、开展其项目活动或者提供服务所发生的费用。
2. **管理费用**：民间非营利组织为组织和管理其业务活动所发生的各项费用。
3. **筹资费用**：民间非营利组织为筹集业务活动所需资金而发生的费用。
4. **其他费用**：除上述费用之外的其他各项费用。

#### 2.6.2 费用确认原则（第42-50条）

1. **业务活动成本确认**：民间非营利组织为开展业务活动所发生的各项费用，应当在同时满足以下条件时确认：
   - 与费用相关的经济利益或者服务潜力很可能流出民间非营利组织；
   - 未来流出的经济利益的金额能够可靠地计量。

2. **管理费用确认**：民间非营利组织为组织和管理其业务活动所发生的各项费用，包括工作人员工资福利、办公费、水电费、物业管理费等。

3. **筹资费用确认**：民间非营利组织为筹集业务活动所需资金而发生的费用，包括利息收入、汇兑损失等。

#### 2.6.3 费用核算科目设置

1. **业务活动成本**：设置"业务活动成本"科目
2. **管理费用**：设置"管理费用"科目
3. **筹资费用**：设置"筹资费用"科目
4. **其他费用**：设置"其他费用"科目

### 2.7 财务会计报告（第51-60条）

#### 2.7.1 财务会计报告构成（第51条）

民间非营利组织的财务会计报告由以下部分构成：
1. **资产负债表**：反映民间非营利组织某一特定日期财务状况的报表
2. **业务活动表**：反映民间非营利组织某一会计期间开展业务活动取得收入、发生费用、实现净资产变动等情况的报表
3. **现金流量表**：反映民间非营利组织某一会计期间现金和现金等价物流入和流出情况的报表
4. **会计报表附注**：对上述报表中列示项目的文字描述或明细资料，以及未能在这些报表中列示项目的说明等

#### 2.7.2 编制要求（第52-55条）

1. **真实性和完整性**：财务会计报告应当真实、完整地反映民间非营利组织的财务状况、业务活动情况和现金流量。
2. **及时性**：民间非营利组织应当按期编制财务会计报告。
3. **审计要求**：民间非营利组织在年检时，应当提供注册会计师审计报告。

#### 2.7.3 年度报告（第56-60条）

1. **年度财务会计报告**：民间非营利组织应当于年度终了后，编制年度财务会计报告。
2. **审计报告**：民间非营利组织应当委托注册会计师对其年度财务会计报告进行审计，并按照规定将审计报告报送登记管理机关。
3. **信息公开**：民间非营利组织应当按照国家法律、行政法规等规定，在其年度检查工作中，向登记管理机关报送年度工作报告、财务会计报告、注册会计师审计报告等。

## 三、实务提示

### 3.1 民间非营利组织设立实务

1. **会计制度选择**：
   - 确认组织性质是否为民间非营利组织
   - 选择适用的会计制度（本制度 vs 《企业会计准则》vs 《事业单位会计制度》）
   - 设置会计科目和会计报表

2. **会计核算基础**：
   - 采用**权责发生制**为基础
   - 会计记账应当采用**借贷记账法**
   - 会计记录文字应当使用**中文**

3. **会计期间划分**：
   - 会计期间至少分为**年度**和**月度**
   - 会计年度、月度等会计期间的起讫日期采用**公历日期**

### 3.2 资产取得和计量实务

1. **货币资产取得**：
   - 按照实际收到的金额入账
   - 设置现金和银行存款日记账，按照业务发生顺序逐日逐笔登记

2. **非现金资产捐赠取得**（重要）：
   - **有凭据**：按照凭据上标明的金额作为入账价值
   - **无凭据或凭据金额与公允价值相差较大**：以公允价值作为入账价值
   - **公允价值无法可靠计量**（固定资产、无形资产捐赠）：按照**名义金额（人民币1元）**入账
   - **服务捐赠**：一般情况下不予确认；提供了发票且能反映公允价值的，按照凭据金额入账

3. **公允价值确定**：
   - 同类或类似资产存在活跃市场的，按照市场价格确定
   - 不存在活跃市场的，采用合理的计价方法确定

4. **资产减值准备计提**：
   - 至少于每年年度终了进行检查
   - 对短期投资、应收款项、存货、长期投资等计提减值准备
   - 对发生减值的固定资产、无形资产等计提减值准备

### 3.3 收入确认和计量实务

1. **捐赠收入确认**（重要）：
   - 确认条件：资产所有权转移 + 无义务提供商品或劳务
   - 分别按照**限定性收入**和**非限定性收入**进行核算
   - 期末结转：限定性捐赠收入结转至限定性净资产；非限定性捐赠收入结转至非限定性净资产

2. **会费收入确认**：
   - 确认条件：签订会费协议 + 会费已收取或能够合理估计
   - 按照会员类型（个人会员、单位会员）设置明细科目

3. **提供服务收入确认**：
   - 确认条件：交易结果能够可靠估计 + 经济利益能够流入 + 收入和成本能够可靠计量
   - 按照服务类型设置明细科目

### 3.4 净资产分类和管理实务

1. **净资产分类**（重要）：
   - **限定性净资产**：存在时间限制或用途限制
   - **非限定性净资产**：无限制

2. **限定性净资产限制解除**：
   - 时间限制已满
   - 用途限制已实现
   - 资源提供者或国家法律、行政法规撤销了限制
   - 限制解除时，将限定性净资产转为非限定性净资产

3. **净资产列报**：
   - 在资产负债表上单独列示限定性净资产和非限定性净资产
   - 在业务活动表上分别反映限定性净资产和非限定性净资产的变动情况

### 3.5 财务会计报告编制和审计实务

1. **财务会计报告构成**：
   - 资产负债表
   - 业务活动表
   - 现金流量表
   - 会计报表附注

2. **编制要求**：
   - 真实性和完整性
   - 及时性（年度终了后编制）
   - 审计要求（委托注册会计师审计）

3. **年度报告报送**：
   - 向登记管理机关报送年度工作报告
   - 年度工作报告包括：财务会计报告、注册会计师审计报告，开展募捐、接受捐赠、提供资助等活动的情况以及人员和机构的变动情况等
   - 按照规定公布年度工作报告，接受社会公众的查询、监督

### 3.6 合规风险提示

1. **会计核算不合规风险**：
   - 未采用权责发生制为基础
   - 未采用借贷记账法
   - 会计记录文字未使用中文
   - 会计政策前后各期不一致且未披露

2. **资产取得和计量不合规风险**：
   - 捐赠资产未取得相关凭据，且未以公允价值入账
   - 公允价值无法可靠计量时，未按照名义金额（1元）入账
   - 未按照规定计提资产减值准备

3. **收入确认不合规风险**：
   - 捐赠收入未分别按照限定性收入和非限定性收入进行核算
   - 收入确认条件未满足时提前确认收入
   - 提供虚假收入信息

4. **净资产分类不合规风险**：
   - 未将净资产按照是否受到限制分为限定性净资产和非限定性净资产
   - 限定性净资产限制已解除，但未进行重新分类
   - 将限定性净资产用于非限定用途

5. **财务会计报告不合规风险**：
   - 财务会计报告不真实、不完整
   - 未按照规定编制现金流量表
   - 未委托注册会计师进行审计
   - 未按照规定报送年度工作报告
   - 未按照规定公布年度工作报告

2. 慈善组织应当每年向其登记的民政部门报送年度工作报告、财务会计报告、注册会计师审计报告
3. 慈善组织应当向社会公布年度工作报告、财务会计报告

**适用要点**：
1. 慈善组织（基金会、社会团体、社会服务机构等）应当执行本制度
2. 慈善组织应当建立健全内部会计监督制度
3. 慈善组织应当每年报送年度工作报告、财务会计报告、注册会计师审计报告
4. 慈善组织应当向社会公布年度工作报告、财务会计报告

### 5.2 实务操作建议

   - 向社会公布年度工作报告、财务会计报告
