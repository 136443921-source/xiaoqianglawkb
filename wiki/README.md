# wiki/ 目录 - 编译后的结构化知识

本目录存放由 AI 编译 raw/ 原始资料后生成的结构化知识页面。

## 子目录说明

- **concepts/** - 法律概念页（如：合同无效、表见代理）
- **entities/** - 实体页
  - `cases/` - 案例页
  - `courts/` - 法院信息页
  - `laws/` - 法律法规页
  - `literature/` - 学术文献页
- **sources/** - 源摘要页（对 raw/ 文档的摘要）
- **comparisons/** - 对比分析页（如：合同无效 vs 合同可撤销）

## 使用说明

1. **不要手动修改本目录中的文件** - 由 AI 根据 AGENTS.md 规范自动维护
2. **查询时优先使用本目录** - 查询工作流先读 index.md，再加载 wiki/ 中的页面
3. **如发现错误** - 可以手动编辑，或删除后重新编译

## 与 raw/ 的关系

- `raw/` = 原材料（只读，AI 不修改）
- `wiki/` = 成品（由 AI 编译生成，结构化、互相关联）
